# Copy Coords Mod

**Copy Coords Mod** - удобный мод для Minecraft, который позволяет копировать координаты игрока в буфер обмена одним нажатием назначенной клавиши.  

Особенности мода:
- Поддержка настраиваемых шаблонов координат (`{x}`, `{y}`, `{z}`, `{world}`).
- Быстрое копирование координат без необходимости открывать чат или команды.
- Настройка через конфиг: можно менять шаблон отображения координат под себя.
- Лёгкий и совместимый с другими модами.

## Установка
1. Скачайте мод и поместите `.jar` файл в папку `mods`.
2. Убедитесь, что установлена совместимая версия Forge.
3. Запустите Minecraft.

## Конфиг
Шаблон координат можно изменить в конфиге
